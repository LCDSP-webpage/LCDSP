Abstract

Language-controlled reinforcement learning (LC-RL) struggles with high-level instructions in complex multi-agent environments like football games. We introduce Language-Controlled Diverse Style Policies (LCDSP), a novel LC-RL paradigm for complex scenarios. LCDSP features two components: Diverse Style Training (DST) that trains a single policy with diverse behaviors via style parameters, and Style Interpreter (SI) that translates language instructions into corresponding parameters. Experiments in 5v5 football demonstrate LCDSP's ability to comprehend tactical instructions and execute diverse behavioral styles effectively.

Diverse style training (DST)

LCDSP enables a single model to be trained with diverse behavioral styles controlled by multi-style parameters. The below figure shows the training process where we design meta-behaviors and establish relationships between style parameters and corresponding behaviors through reward shaping.

As instruction complexity increases, more style parameters are needed, but this significantly increases training difficulty. Current MORL methods typically support only 2-4 objectives, while our GRF 5v5 environment incorporates 10 meta-behaviors. To address this challenge, we propose Prioritized Style Sampling (PSS), which focuses on training more valuable style parameters using entropy-based criteria to enhance training efficiency.

During training, the style generator produces style parameters that dictate reward magnitudes for associated meta-behaviors. The agent receives environment state and style parameters to generate actions, while the environment provides rewards and next states for RL training. This data also adjusts the style generator's probability distribution. The second figure above shows two different styles of a trained policy in the GRF 5v5 scenario.

LCDSP inference process

Aligning language instructions with target policies

During inference, the Style Interpreter (SI) processes instructions to generate corresponding Style Parameters (SP). The SI addresses the challenge of translating high-level instructions into specific parameter values by understanding each parameter's meaning and determining relevant values.

The SI is implemented by fine-tuning a network built upon a pre-trained language model (PLM). User instructions are processed by the PLM to obtain representations, which serve as input for the Adaptive Style-adjustment Block (ASaB) responsible for generating SP.

The ASaB incorporates behavior descriptions processed by the same PLM to generate adaptive scaling parameters $\gamma_i$ and $\beta_i$ for each style parameter. It then applies an affine transformation: $$\text{ASaB}(Y_i | \gamma_i, \beta_i) = \gamma_i \cdot Y_i + \beta_i$$ where $Y_i$ represents the initial output for the i-th style parameter. The network uses multiple output heads for a multi-head regression task to predict the ground truth SP vector $\omega$.

Overview of the SI module

High-Level Instructions Following Performance

We evaluate LCDSP on the Google Research Football (GRF) 5v5 scenario, a complex multi-agent environment with 3K-step episodes and sparse rewards. To test high-level instruction following, we designed six football tactics: Positive Attack, All-out Attack, Balanced Play, Counter Attack, Park the Bus, and Tiki-Taka.

We generated 30 instructions per tactic (180 total) with corresponding strategy descriptions. Results show LCDSP accurately comprehends tactical instructions and executes diverse behavioral styles, demonstrating strong generalization across different instruction translation methods.
